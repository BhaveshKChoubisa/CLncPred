CLnc-Pred User Guide

Getting Started

CLnc-Pred is a machine learning tool for predicting long non-coding RNAs (lncRNAs) in plant genomes. Follow the steps below to use the tool effectively.

Step 1: Upload or Paste Sequence

You can either:

  • Paste a sequence in FASTA format.
  • Upload a FASTA file (.fasta, .fa).

Step 2: Click Predict

After entering the sequence, click the Predict button to analyze the data.

Step 3: View Results

The prediction results will be displayed, indicating whether the sequence is a lncRNA or coding RNA with probability. The results can be downloaded by download button
